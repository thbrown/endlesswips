# Endless WIPS

## Install

npm install -g gatsby-cli
npm install -g typescript
npm install

## Build and Run

npm run-script:
"develop": "gatsby develop",
"build": "gatsby build",
"start": "npm run develop",
"serve": "gatsby serve",
"clean": "gatsby clean",

e.g. npm run-script develop

./release.sh to move build files and commit them
